# generals.io toolkit
This is a userscript to help with your games at [generals.io](https://generals.io/), which is loaded using TamperMonkey. Features of this script include:
* Make the background brighter so you can see more clearly
* Count the number of cities your opponent has
* Remember the location of kings and cities you've seen earlier
* Automatically take over control and rush into enemy's king if your army is adjacent to his king and the number of soldiers is enough
